Random Task Generator (Генератор случайных задач)

GUI-приложение «Random Task Generator» с использованием библиотеки random, сохранением истории и Git.

Реализовано:
*хранение задач в json
*добавление новых задач
*редактирование задач
*удаление задач

Для запуска используйте команду python main.py
